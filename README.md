# 🌍 RoamLanka

RoamLanka is an Android-based travel guide application designed to help local and international travelers explore tourist attractions in Sri Lanka, plan trips, view locations on Google Maps, check weather updates, and share travel experiences.

---

## 📱 Features

- 🔐 User Registration & Login (Firebase Authentication)
- 📍 Explore Tourist Attractions
- 📝 Trip Planning & Saving Places
- 🗺 Google Maps Integration (Location & Routes)
- 🌦 Weather Information (OpenWeather API)
- 💬 Travel Stories Sharing
- ❓ Community Q&A (Ask & Answer)
- 👤 User Profile Management

---

## 🛠 Technologies Used

### Frontend:
- Android Studio
- Java
- XML Layouts
- RecyclerView
- CardView
- Material Design Components

### Backend:
- Firebase Authentication
- Firebase Firestore
- SharedPreferences

### APIs:
- Google Maps API (Location display, markers, routes)
- Geocoder API (Location conversion)
- OpenWeather API (Weather updates)

---

## 🗄 Database Structure (Firestore)

- Users
- SavedPlaces
- StartLocation
- Stories
- Comments
- Community
- Answers

## 🎯 Objectives
